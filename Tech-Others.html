Computer Hardware

Computer Science is perhaps the fastest growing industry in the 21st century its applications ranging from trivial everyday tasks to high tech international banking transactions. This course provides an overview of computer hardware, providing a basic understanding of all the various components of a computer system. Graduates of this course will learn how to set up, configure and troubleshoot a basic computer system.

Computer hardware includes the physical parts or components of a computer, such as the central processing unit, monitor, keyboard, computer data storage, graphic card, sound card and motherboard. By contrast, software is instructions that can be stored and run by hardware. Hardware is directed by the software to execute any command or instruction.

A combination of hardware and software forms a usable computing system. This course also creates a strong foundation in individuals and makes it easy for them to showcase it in a platform of their choice.

Computer Networking

In the modern age all computers in any kind of office or bank are networked to one another. We provide different courses on networking depending on the needs of the individual. The course can be about the basic concepts in networking or a detailed study of all the concepts in CCNA. CCNA (Cisco Certified Network Associate) is an information technology (IT) certification from Cisco. This course provides the skills required to install, operate, and troubleshoot a small to medium size enterprise branch network.

The topics include connecting to a WAN; implementing network security; network types; network media; routing and switching fundamentals; the TCP/IP and OSI models; IP addressing; WAN technologies; operating and configuring IOS devices; extending switched networks with VLANs; determining IP routes; managing IP traffic with access lists; establishing point-to- point connections; and establishing Frame Relay connections.

Eligibility: Plus Two or Any Degree
Duration: 2 Months
